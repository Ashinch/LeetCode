#### 介绍

校内实训网站学习作品，以[LeetCode](https://leetcode.com/)网站为主要参考，加以想法重新设计了网站界面和新增功能。

#### 软件环境

* MyEclipse 8.5
* JDK 1.6
* MS SQL SERVER 2008 R2

#### 项目架构
* jsp
* spring+springmvc+mybatis (SSM) 框架
